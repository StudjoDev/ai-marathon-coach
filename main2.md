# Prompt 17：賽事資料型別與工具函式

```md
請新增 src/utils/raceUtils.ts

實作以下函式：

1. getUpcomingRaces(races, today)
回傳尚未完成賽事，依日期排序

2. getNextRace(races, today)
回傳最近一場賽事

3. getDaysUntilRace(date, today)
回傳倒數天數

A -> A 賽｜年度目標
B -> B 賽｜測驗賽
C -> C 賽｜輔助賽

5. getRaceCategoryStyle(category)
回傳對應 Tailwind className：
A：主色強調
B：藍綠色
C：灰紫色

6. getRaceStrategyById(raceId)
回傳該場比賽策略陣列

請補上 TypeScript 型別，不要使用 any。
```

```txt
- A/B/C定位
- 詳情 Bottom Sheet

- 趨勢圖
- 跑後回饋

疼痛

- 疼痛位置
- 降載提醒

教練
- AI 分析
- 本週策略
- 下週是否升級
```

---

最重要的是：
**不要把賽事資訊做成一張大表。**

手機 App 裡應該用：

> 下一場賽事倒數 + 時間軸 + 詳情 Bottom Sheet

這樣跑者才會真正看得懂、用得起來。
